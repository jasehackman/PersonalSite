Song-Share

Song-Share is a single page responsive CRUD web application built with React.js and Bootstrap that reduces the barriers between publishers and songwriters. Song-Share allows songwriters to upload songs, build playlists and securely share those playlists with a publisher. A publisher can then hear each song on the playlist in one page and read pertinent information on each song.

Mission Control is a single page responsive CRUD web application built with JavaScript. Mission Control was designed to bring several aspects of online interaction together in one application. In Mission Control a user can manage their tasks, store articles, manage their events, add friends and allow the user to message other users. I had a role in all aspects of this project but was predominantly responsible for the events page.
